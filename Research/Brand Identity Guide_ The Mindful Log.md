# Brand Identity Guide: The Mindful Log

## 1. Brand Vision
**The Mindful Log** is an AI-powered psychology journal designed to bridge the gap between clinical psychology and daily reflection. It provides a "reflective mirror" to help users identify cognitive distortions and emotional patterns in a safe, whimsical, and non-judgmental space.

## 2. Visual Aesthetic: The "Doodles" Influence
The brand draws inspiration from the **Doodles NFT** collection by **Burnt Toast (Scott Martin)**. The aesthetic is characterized by:
- **Playful Minimalism:** Simple, hand-drawn line art with bold, consistent strokes.
- **Whimsical Motifs:** Hand-drawn clouds, smiling planets, and "left eyes" drawn outside face boundaries.
- **Friendly Characters:** Soft, rounded shapes that evoke comfort and safety.

## 3. Color Palette
A pastel-heavy palette that feels soothing yet vibrant.

| Color Name | Hex Code | Visual Reference |
| :--- | :--- | :--- |
| **Pastel Pink** | `#FFADDE` | Primary Brand Color |
| **Pastel Blue** | `#ACBEFC` | Secondary Brand Color |
| **Mint Green** | `#95F0C5` | Action / Success Color |
| **Soft Lavender** | `#C5C5FF` | Accent Color |
| **Mellow Yellow** | `#FFD2EA` | Highlight Color |
| **Peach Fuzz** | `#FFCC99` | Warmth Color |

## 4. Typography
The typography should be rounded and approachable to match the "Doodles" aesthetic.

- **Primary Font:** `Quicksand` (Sans Serif, Rounded)
  - *Usage:* Headings, UI Buttons, Navigation.
- **Secondary Font:** `Comfortaa` or `Varela Round`
  - *Usage:* Body text, journal entries, long-form reading.

## 5. Brand Voice & Tone
- **Supportive:** Like a warm hug after a long day.
- **Whimsical:** Finding joy in small moments and self-discovery.
- **Non-Judgmental:** A safe space where every emotion is valid.
- **Insightful:** Using AI not to "fix" the user, but to offer a clear, objective mirror.

## 6. Visual Motifs
- **The "Reflective Mirror":** A recurring motif of a hand-held mirror with a smiling face.
- **Doodle Elements:** Small, floating stars, clouds, and abstract "blobs" in the UI background.
- **Emotional Icons:** Custom-drawn icons for different moods (e.g., a cloud for "High Distress", a sun for "Positive").
